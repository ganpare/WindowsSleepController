Setup Instructions
1. Generate an API Key

Click the "Generate New Key" button above to create an API key. Save this key securely as it will only be shown once.

2. Configure Alexa Skill

Create a custom Alexa skill using AWS Lambda that sends a request to your endpoint with the API key.

3. Configure Your Network

Make sure port 5000 is forwarded to this computer on your router, or use a service like ngrok to expose your endpoint securely.

4. Test Your Setup

Try sending a request to your endpoint using the API key, or say "Alexa, tell my computer to sleep" (depending on your skill configuration).
